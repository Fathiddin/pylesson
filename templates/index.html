Python nima ?
Python ommabop dasturlash tili. Uni 1991-yil Guido van Rossum ishlab chiqgan.
Python quyidagi sohalarda ko'p ishlatiladi:
-web dasturlash (serverlar bilan);
-dasturiy ta'minot;
-matematika;
-tizim scriptlari
Nega Python ?
Python turli platmorfalarda ishlaydi (Windows, Mac, Linux, Raspberry Pi va boshqalar);
-Python sodda sintaksisga ega;
-Sodda sintaksis tufayli boshqa dasturlash tillariga nisbatan dasturlashda kamroq kod ishlatiladi;
-Pythonning amallarini bajarish tezligi yuqori;
Saytimizda!
